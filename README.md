# 📊 Customer Segmentation Project

## 📷 Dashboard Preview
![Dashboard](Screenshot%202026-05-05%20215642.png)

## 📌 Overview
This project focuses on segmenting customers based on their behavior and purchasing patterns. The goal is to identify different customer groups and provide insights for better business decision-making.

## 🎯 Objectives
- Analyze customer data
- Identify high-value customers
- Group customers into segments
- Improve marketing strategies

## 🛠️ Tools Used
- Power BI
- Excel

## 📈 Key Insights
- Identified different customer segments based on behavior
- Found high-value and low-value customers
- Helped in targeting the right audience
- Improved understanding of customer patterns

## 🚀 Conclusion
Customer segmentation helps businesses understand their customers better and make data-driven decisions for growth.
